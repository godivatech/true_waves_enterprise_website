**ROLE & OBJECTIVE**
Act as a Senior Product Designer and Lead Frontend Engineer from a top-tier enterprise company.

Design and build a **production-ready, high-trust, conversion-focused website** for a professional security firm named **True Waves Enterprises**.

Tech stack:

* Next.js (App Router)
* Tailwind CSS
* Framer Motion

The output must feel like a **real company website**, not a UI experiment or template.

---

# 🔒 CORE PRINCIPLES (NON-NEGOTIABLE)

* Do NOT generate a generic template layout
* Do NOT overdesign or add decorative UI
* Do NOT mix multiple visual styles
* Every section must have a clear business purpose
* Focus on trust, authority, and clarity

👉 Guiding rule:
Design for a user making a serious security decision.

---

# 🎨 COLOR & BACKGROUND SYSTEM (CRITICAL)

Use a **hybrid dark + light system**:

Dark sections:

* `bg-slate-950` (primary)
* `bg-slate-900` (alternate)

Light sections:

* `bg-white`
* `bg-slate-50`

Rules:

* NEVER use a single background for the full page
* Alternate sections for visual hierarchy
* Use dark for authority (hero, branding)
* Use light for readability (content sections)

Accent:

* Green → ONLY for primary CTAs
* Deep blue/cyan → headings or highlights

---

# 📐 LAYOUT SYSTEM

* Container: `max-w-7xl mx-auto px-6`
* Grid: 12-column responsive grid
* Section spacing: `py-16 md:py-24`
* Strict spacing consistency (8px system)

---

# 🧩 COMPONENT RULES

Buttons:

* One primary CTA per section (solid green)
* Secondary = outline/ghost
* No multiple competing CTAs

Cards:

* Flat design
* Subtle borders only
* Dark: `border-white/10`
* Light: `border-gray-200`
* No glassmorphism or heavy shadows

Icons:

* Use `lucide-react`
* Keep consistent stroke width

---

# ✍️ TYPOGRAPHY

* Font: Inter / modern sans-serif
* Bold, strong headings
* Clear hierarchy (headline → subtext → action)
* Avoid long paragraphs
* Optimize for scanability

---

# 🎞️ ANIMATION

Use ONLY:

* Fade-in
* Slight upward motion (y: 20 → 0)
* Staggered reveal

Avoid:

* Complex or flashy animations

---

# ⚠️ NEGATIVE PROMPTING (IMPORTANT)

Do NOT:

* Use random gradients
* Add glow effects
* Create fancy UI without purpose
* Make it look like a template marketplace design
* Overuse icons or visuals

---
